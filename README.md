# Activity Scheduler + Tracker

A daily activity scheduler and tracker. Define activities, schedule them across your week, check in nightly, and track your progress with weekly reports and achievements.

## Features

- **Activity Library** - Create and manage activities with icons, colors, and categories
- **Calendar** - Week and day views with drag-and-drop scheduling
- **Sets** - Bundle activities together for quick scheduling
- **Presets** - Save and reuse weekly schedule templates
- **Nightly Check-in** - Daily reminders to mark activities as done/missed
- **Weekly Reports** - Track completion rates and insights
- **Proof Feed** - Attach photos and comments to completed activities
- **Achievements** - Earn badges for consistency and milestones

## Tech Stack

- **Frontend**: React Native / Expo
- **Backend**: Appwrite (auth, database, storage, functions)
- **State Management**: TanStack Query
- **Notifications**: expo-notifications, FCM
- **Charts**: react-native-svg, victory-native

## Getting Started

```bash
# Install dependencies
npm install

# Run the app
npm start
```
