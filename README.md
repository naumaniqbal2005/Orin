# Activity Scheduler + Tracker

A daily activity scheduler and tracker. Define activities, schedule them across your week, check in nightly, and track your progress with weekly reports and achievements.

## Features

- **Activity Library** - Create and manage activities with icons, colors, and categories
- **Calendar** - Week and day views with drag-and-drop scheduling
- **Sets** - Bundle activities together for quick scheduling
- **Presets** - Save and reuse weekly schedule templates
- **Nightly Check-in** - Daily reminders to mark activities as done/missed
- **Weekly Reports** - Track completion rates and insights
- **Proof Feed** - Attach photos and comments to completed activities
- **Achievements** - Earn badges for consistency and milestones

## Tech Stack

- **Frontend**: React Native / Expo
- **Backend**: Appwrite (auth, database, storage, functions)
- **State Management**: TanStack Query
- **Notifications**: expo-notifications, FCM
- **Charts**: react-native-svg, victory-native

## Getting Started

```bash
# Install dependencies
npm install

# Run the app
npm start
```

UI/UX Flow

Use a pastel UI approach to create a calm, welcoming, and emotionally supportive feel. The design should rely on soft gradients, rounded shapes, and airy spacing rather than sharp contrasts or heavy blocks.

1. **Color Palette**  
   - Choose high-lightness, low-saturation tones such as lavender, blush pink, mint, sky blue, or cream.  
   - Blend colors with white to keep them soft and avoid harshness.  
   - Apply subtle gradients (e.g., purple → pink, mint → cream) for depth without strong contrast.

2. **Shapes and Layout**  
   - Use rounded corners consistently across cards, buttons, and icons.  
   - Keep spacing generous to maintain an airy, uncluttered feel.  
   - Apply soft shadows that are diffuse, giving elements a floating, lightweight appearance.

3. **Typography and Icons**  
   - Select a clean sans-serif font. Use medium weight for titles and lighter weight for supporting text.  
   - Keep text sizes balanced — large enough for clarity but not bold or heavy.  
   - Use minimalist line icons, outlined rather than filled, to match the light aesthetic.

4. **Cards and Panels**  
   - Design cards with semi-transparent or lightly tinted pastel backgrounds.  
   - Layer them over gradient backdrops to create depth.  
   - Maintain consistent rounded edges and soft shadows.  
   - Size cards proportionally: larger for primary focus, smaller for secondary, but all visually cohesive.

5. **Overall Feel**  
   - The interface should feel calm, approachable, and emotionally supportive.  
   - Avoid sharp contrasts, dark tones, or rigid geometry.  
   - Strive for clarity and friendliness, making the app feel like a companion rather than a tool.
